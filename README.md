# A simple application to practice Interfaces and Generics in Go

- Navigate into the application folder on the command line and run the application using `go run .`
  - It displays the car and bus details already defined in the code
    - This is to showcase interfaces related features
  - It also shows the results of Add and Square methods
    - This is to showcase the generics related features
- The application has 3 packages.
  - The default `main` package
  - `vehicle` package for all implementation related to interfaces
  - `generics` package for all implementation related to generics
